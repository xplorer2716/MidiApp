# MidiApp
A simple framework to make .NET 8.0+ desktop midi apps, based on the updated Leslie Sanford's C# MIDI Toolkit (see https://github.com/xplorer2716/Sanford.Multimedia.Midi)


## Build Instructions

### Prerequisites
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) or later
- Visual Studio 2022 (recommended) or any IDE supporting .NET development


### Steps to Build
1. Clone the https://github.com/xplorer2716/Sanford.Multimedia.Midi repo
2. Clone this repo https://github.com/xplorer2716/MidiApp to the same parent directory as the Sanford repo.
3. Open MidiApp.MidiController.sln in Visual Studio and build.
